## Mini Gifs Index Web App
1.  Show list trending gif
1.  Search gif
1.  Infinite Scroll

## Tech Stack

1. React
1. Redux 
1. Axios
1. NodeJs
1. NPM
1. Yarn

### Install Package

```
npm install / yarn install
```

### Setup Development Mode

```
npm start / yarn start
```
